# What is Strange Homeworlds

Strange Homeworlds is an addon to the mod [Colonize UnlimitedÂ©](https://steamcommunity.com/sharedfiles/filedetails/?id=682582076) and is designed to let you start with previously uninhabitable planets!

Get it on Steam Workshop: http://steamcommunity.com/sharedfiles/filedetails/?id=716723174

# Features
Ability to start out with the following world types (in addition to the usual ones):
* Molten
* Toxic
* Barren
* Barren Cold
* Frozen
* Gas Giant
* Tomb World
* AI World
* Space Habitats
* Shielded World
